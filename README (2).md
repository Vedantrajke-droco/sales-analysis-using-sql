# Data Filtering, Sorting and Aggregation

## Task Overview
This task focuses on practicing basic SQL concepts using a sales dataset. The goal is to understand how data can be filtered, sorted, and summarized using SQL queries, which are commonly used in data analysis.

---

## Tools & Environment
- SQLite Database
- Command Prompt (CMD)

---

## Dataset Information
- Dataset Name: Sales Data
- File: `sales_data_task3.csv`
- Total Records: 100+
- Fields:
  - Order_ID
  - Order_Date
  - Customer_Name
  - Category
  - Region
  - Sales
  - Profit

---

## SQL Concepts Applied

### Data Inspection
- Displayed sample records
- Verified total row count

### Filtering Data
- Used `WHERE` clause to filter records based on category and region

### Sorting Data
- Sorted records using `ORDER BY` to find highest sales

### Aggregation
- Calculated total sales by category
- Calculated average profit by region
- Counted total number of orders

### Group Filtering
- Used `HAVING` clause to filter aggregated results

### Advanced Queries
- Used `BETWEEN` to filter records by date range
- Used `LIKE` to search customer names
- Found top 5 customers based on total sales
